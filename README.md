## ✨ Features

| Feature | Description |
|--------|-------------|
| 🎬 **Movie Discovery** | Browse trending, popular, and top-rated movies pulled live from the API |
| 🔍 **Search** | Instant search across thousands of movies by title |
| ⭐ **Ratings & Info** | View ratings, release dates, genres, and overviews per movie |
| 🎭 **Dynamic Content** | All data is live — always up to date |
| 📱 **Responsive** | Works seamlessly across desktop and mobile |
| 🌙 **Dark UI** | Cinematic dark interface built on the DarkNova design system |
